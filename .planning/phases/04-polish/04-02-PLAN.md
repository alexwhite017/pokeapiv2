  <action>
    **Part A — App.css: Add shimmer keyframe**

    In `src/App.css`, after the `#root` block at the bottom, add a custom keyframe animation and utility class for the shimmer effect:

    ```css
    @keyframes shimmer {
      0%   { background-position: -200% center; }
      100% { background-position:  200% center; }
    }

    .animate-shimmer {
      background: linear-gradient(
        90deg,
        #27272a 25%,
        #3f3f46 50%,
        #27272a 75%
      );
      background-size: 200% 100%;
      animation: shimmer 1.5s ease-in-out infinite;
    }
    ```

    This creates a shimmer that sweeps left-to-right using zinc-800 (#27272a) as base and zinc-700 (#3f3f46) as the highlight band. The `background-size: 200%` makes the gradient wider than the element so the sweep is visible.

    ```jsx
        </div>

    };

    export default SkeletonCard;
    ```

    - All shimmer elements use `animate-shimmer` class defined in App.css
    - No props needed — all skeletons are identical (type is unknown)
