Create `src/components/PokemonCard.jsx` — a standalone card component implementing the Phase 2 design.

Props:
```js
// PokemonCard({ name, id, primaryType, secondaryType })
// name: string (e.g. "bulbasaur")
// id: number (e.g. 1)
// primaryType: string (e.g. "grass")
// secondaryType: string | null (e.g. "poison" or null)
```

Card structure (top to bottom):
1. **Card wrapper** — Link to `/details/{name}`, full card clickable
2. **Artwork area (~65–70% of card height)** — official artwork on type gradient background
3. **Dex number** — `#NNN` zero-padded to 3 digits, pinned top-right corner of the CARD (absolute positioned), outside the info strip
4. **Info strip (~30–35% of card height)** — name on top, type badge(s) below

**Gradient background implementation:**
Use inline style for the gradient (NOT dynamic Tailwind gradient utilities) to avoid Tailwind JIT issues with runtime values:
```jsx
style={{ background: `linear-gradient(to bottom, var(--color-${primaryType}), var(--color-surface-base))` }}
```
This uses CSS custom properties directly (the tokens are defined as `--color-{type}` in App.css @theme).

**Card wrapper classes:**
```
relative rounded-xl overflow-hidden cursor-pointer
transition-transform transition-shadow duration-150 ease-out
hover:scale-[1.03] hover:shadow-2xl
```
The card and artwork transform together as one unit. No independent artwork animation.

**Artwork area:**
```jsx
<div className="relative flex items-center justify-center p-3" style={{ height: '65%' }}>
  <img
    src={`https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/${id}.png`}
    alt={name}
    className="w-full h-full object-contain drop-shadow-lg"
  />
</div>
```

**Dex number (absolute top-right):**
```jsx
<span className="absolute top-2 right-2 text-xs font-bold text-white/70 tabular-nums">
  #{String(id).padStart(3, '0')}
</span>
```

**Info strip:**
Use a semi-transparent dark overlay for the info strip so the gradient shows through slightly at the top of the strip, blending gracefully:
```jsx
<div className="bg-black/40 px-3 py-2 flex flex-col items-center gap-1" style={{ height: '35%' }}>
  <span className="text-white font-bold text-sm capitalize leading-tight">
    {name}
  </span>
  <div className="flex gap-1">
    {/* primary type badge */}
    <span className={`bg-${primaryType} text-white text-xs font-semibold px-2 py-0.5 rounded-full capitalize`}>
      {primaryType}
    </span>
    {/* secondary type badge — only if it exists */}
    {secondaryType && (
      <span className={`bg-${secondaryType} text-white text-xs font-semibold px-2 py-0.5 rounded-full capitalize`}>
        {secondaryType}
      </span>
    )}
  </div>
</div>
```

**Text contrast note (from STATE.md blocker):** Electric, Ground, and Ice types are bright colors. Use `text-white` universally — the badge background is the type color which provides sufficient contrast framing even for bright types. Do NOT add per-type text color logic.

**Full component structure:**
```jsx
import { Link } from "react-router";

const PokemonCard = ({ name, id, primaryType, secondaryType }) => {
  return (
    <Link
      to={`/details/${name}`}
      className="block h-full"
    >
      <div
        className="relative rounded-xl overflow-hidden cursor-pointer transition-transform transition-shadow duration-150 ease-out hover:scale-[1.03] hover:shadow-2xl h-full flex flex-col"
        style={{ background: `linear-gradient(to bottom, var(--color-${primaryType}), var(--color-surface-base))` }}
      >
        {/* Dex number — absolute top-right */}
        <span className="absolute top-2 right-2 text-xs font-bold text-white/70 tabular-nums">
          #{String(id).padStart(3, '0')}
        </span>

        {/* Artwork area */}
        <div className="flex-1 flex items-center justify-center p-3 min-h-0">
          <img
            src={`https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/${id}.png`}
            alt={name}
            className="w-full h-full object-contain drop-shadow-lg max-h-36"
          />
        </div>

        {/* Info strip */}
        <div className="bg-black/40 px-3 py-2 flex flex-col items-center gap-1">
          <span className="text-white font-bold text-sm capitalize leading-tight">
            {name}
          </span>
          <div className="flex gap-1 flex-wrap justify-center">
            <span className={`bg-${primaryType} text-white text-xs font-semibold px-2 py-0.5 rounded-full capitalize`}>
              {primaryType}
            </span>
            {secondaryType && (
              <span className={`bg-${secondaryType} text-white text-xs font-semibold px-2 py-0.5 rounded-full capitalize`}>
                {secondaryType}
              </span>
            )}
          </div>
        </div>
      </div>
    </Link>
  );
};

export default PokemonCard;
```

NOTE: The gradient uses inline `style` with CSS custom properties (`var(--color-${primaryType})`) rather than dynamic Tailwind gradient class interpolation — this avoids the Tailwind JIT purge problem and follows the established pattern from StatGraph.jsx (inline style for runtime-computed values, per plan 01-02 decision).
