For alternating row contrast on a dark page, the two-tier approach:
- `even:bg-surface-inset` → zinc-700 — the slightly lighter alternate row
- `odd:bg-surface-raised` → zinc-800 — matches the table base, creates the even/odd rhythm

This pairing gives a subtle dark alternating stripe effect without any light surfaces.
